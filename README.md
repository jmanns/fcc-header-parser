# Free Code Camp Request Header Parser Microservice Project

### User Story

1) I can get the IP address, language and operating system for my browser

### Example Usage

````
https://fcc-header-parser-app.herokuapp.com/api/whoami
````

### Example Output

````json
{
  "ip": "51.37.102.10",
  "os": "Windows 10",
  "lang": "en-GB"
}
````
